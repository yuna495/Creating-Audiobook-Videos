# 朗読動画作成アプリ

音声ファイルとテキスト字幕、背景画像、BGMを統合し、美しい演出効果を伴う朗読動画を自動生成するデスクトップアプリケーションです。

## 主な機能

- **GUI設定ツール**
  - 対象作品フォルダの選択、タイトル、作者クレジット、デフォルト音声間隔などの設定。
  - 音声リストと個別インターバル（秒）を調整できるスクロールエリア。マウスホイールによるスクロールに対応し、スライダーと数値入力での直感的な調整が可能。
  - 背景画像やBGMの適用範囲を、ドロップダウンメニューから感覚的に指定可能。BGMは適用区間のミュートや音量設定、自動フェードアウトにも対応。
  - 設定内容は自動・手動で保存され、次回起動時に復元可能。
  - リアルタイムで進行状況を確認できる詳細な実行ログ表示エリア。
- **動画生成エンジン ([generator.py](file:///d:/Games/Python/朗読動画/朗読動画作成アプリ/generator.py))**
  - タイトルとクレジット表示用のアニメーション（イージング、グロー、ガウスぼかしによるフェードイン/フェードアウト）。
  - 日本語の自動改行、ボックス背景、シャドウが付いた読みやすい字幕描画。
  - 背景画像の「Cover（アスペクト比維持の画面フィット）」配置と、シーン切り替え時のスムーズなクロスフェード効果。
  - FFmpegを活用した高品質な音声ミックス・動画エンコード処理。

## フォルダ構成の例

対象の作品フォルダは、以下の構成にすることを想定しています。

```
[作品フォルダ]
│  project_config.json  # プロジェクトごとの設定（自動生成・保存）
│  字幕.json            # 各音声に対応する字幕テキスト（{"003": "字幕本文..."}）
│
├─音声ファイル          # 音声ファイルを格納するフォルダ
│  │  001_xxx.wav       # タイトル朗読用音声（ファイル名先頭が 001）
│  │  002_xxx.wav       # 作者クレジット朗読用音声（ファイル名先頭が 002）
│  │  003_xxx.wav...    # 本文の音声ファイル（ファイル名先頭が 003以降）
│  │
│  └─BGM                # BGMファイルを格納するフォルダ（任意）
│          bgm1.wav
│          bgm2.wav
│
└─背景画像              # 背景画像を格納するフォルダ
        003~010.png     # 音声インデックスの適用範囲をファイル名にした画像
        011~022.png
```

## 動作環境・必要ライブラリ

本アプリの動作には Python 3.x が必要です。

### 依存パッケージのインストール

以下のパッケージをインストールしてください。

```bash
pip install numpy Pillow imageio imageio-ffmpeg
```

※ 日本語フォントとして、Windows 標準の游明朝（YuMincho）や MS明朝、MSゴシックなどが自動的に探索され使用されます。

## 起動方法

アプリのディレクトリに移動し、以下のコマンドで起動します。

```powershell
python gui.py
```
